# 单例模式

保证一个类只有一个实例，并且提供一个全局访问点。

常用在重量级的实例对象中，不需要多个实例，如线程池，数据库连接池中等。

```
,-------------------------.
|Singleton                |
|-------------------------|
|-singleton: Singleton    |
|-------------------------|
|-Singleton()             |
|+getInstance(): Singleton|
`-------------------------'

```
